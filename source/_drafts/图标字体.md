## 安装

``` bash
yarn add react-native-svg
```

因为涉及到原生代码, 需要重新安装应用

``` bash
# 重新安装 ios
cd ios
pod install
npm run ios

# 重新安装 andorid
npm run android
```

``` bash
yarn add -D react-native-iconfont-cli

# 创建配置文件
npx iconfont-init
```

根据需求配置 `iconfont.json`
``` json
{
    "symbol_url": "http://at.alicdn.com/t/font_2018804_v2ssfx4wlub.js",
    "use_typescript": true,
    "save_dir": "./src/components/iconfont",
    "trim_icon_prefix": "",
    "default_icon_size": 18
}
```

## 生成组件
``` bash
npx iconfont-rn
```
